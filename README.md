# Datenschutz-und-Datensicherheit-Quiz
Quiz über Datenschutz und Datensicherheit

LF4 Datenschutz & Datensicherheit – Quiz und Lernmodus

So startest du die Website:
1. ZIP-Datei entpacken.
2. index.html doppelt anklicken.
3. Die Seite läuft komplett offline im Browser.

Funktionen:
- Kapitel-Auswahl am Anfang
- Eigener Quizmodus pro Kapitel
- Sofortanzeige nach jeder Frage: richtig/falsch
- Tipps, die nicht direkt die Lösung verraten
- Zurück-Button im Quiz
- Lösungen am Ende mit Erklärung
- Lernmodus mit Suchfunktion und Kapitel-Filter
- Dark Mode
- Saving System über LocalStorage im Browser

Wichtig zum Saving System:
Der Fortschritt wird lokal im Browser gespeichert. Das bedeutet:
- funktioniert ohne Server und ohne Login
- bleibt im gleichen Browser erhalten(Über Cookies)
- wird nicht zwischen verschiedenen Geräten synchronisiert
- kann über "Speicher löschen" zurückgesetzt werden

Für Cloud-Speicherung bräuchte man zusätzlich einen Server mit Datenbank/Login.
